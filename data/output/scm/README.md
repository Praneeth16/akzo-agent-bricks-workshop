# SCM (Supply Chain) Synthetic Data — AkzoNobel Coatings Workshop

Generated by [`data/generate_scm.py`](../../generate_scm.py) with a **fixed seed (43)**, so
output is fully deterministic and reproducible. Tables share the same
dimensional grain (`sku` x `region` x `month`) as the finance generator and
reuse the **exact finance `sku` ids** (read from
`data/output/finance/products.parquet` when present), so SCM joins cleanly to
finance for cross-domain demos.

Libraries: stdlib + numpy + pandas + pyarrow only.

## Dimensions

| Dimension | Values |
| --- | --- |
| Time | 24 monthly periods, `month` = first-of-month `DATE`, `2024-07-01` .. `2026-06-01` (current = `2026-06`) |
| Regions | EMEA, Americas, APAC, China |
| Plants (6) | Rotterdam-NL (EMEA), Felling-UK (EMEA), Houston-US (Americas), Shanghai-CN (China), Pune-IN (APAC), Sao-Paulo-BR (Americas) |
| Lanes (11) | origin_plant -> dest_region; modes road / sea / air |
| SKUs | ~60 coatings SKUs reused from finance (`DEC-####` Decorative Paints, `PERF-####` Performance Coatings). Fallback `SKU0001..SKU0060` if finance products are absent. |

## Tables

All written as parquet (pyarrow, no index) to `data/output/scm/`.

### `lanes.parquet` (11 rows) — lane master

| Column | Type | Description |
| --- | --- | --- |
| `lane_id` | string | e.g. `Rotterdam-NL->EMEA-DACH` |
| `origin_plant` | string | one of the 6 plants |
| `dest_region` | string | destination sub-region label (e.g. `EMEA-DACH`) |
| `mode` | string | `road` / `sea` / `air` |
| `lead_time_days` | int | **current (Q2 2026)** transit lead time; narrative lane shows the elevated value |
| `cost_per_unit` | float | EUR per unit shipped on the lane |

### `otif.parquet` (1,320 rows) — On-Time-In-Full by plant x region x lane x sku x month

| Column | Type | Description |
| --- | --- | --- |
| `plant` | string | origin plant |
| `region` | string | dimension region (EMEA / Americas / APAC / China) |
| `lane` | string | `lane_id` |
| `sku` | string | finance-aligned SKU |
| `month` | date | first-of-month |
| `orders` | int | total order lines in the month |
| `on_time` | int | order lines delivered on time (>= perfect) |
| `in_full` | int | order lines delivered in full (>= perfect) |
| `otif_pct` | float | perfect orders / orders, where perfect = orders both on-time AND in-full |

**Invariant:** `otif_pct = perfect / orders` where `perfect = min(on_time-overlap, in_full-overlap)`,
so `otif_pct <= on_time/orders` and `otif_pct <= in_full/orders` for every row
(verified at generation time; `otif_pct` is truncated to 4 decimals to never
round above the true ratio).

### `inventory.parquet` (1,320 rows) — inventory health by plant x sku x month

| Column | Type | Description |
| --- | --- | --- |
| `plant` | string | stocking plant |
| `sku` | string | finance-aligned SKU |
| `month` | date | first-of-month |
| `on_hand_units` | int | units on hand |
| `safety_stock` | int | target safety-stock level |
| `days_of_supply` | float | `on_hand_units / daily_demand` |
| `stockout_flag` | int | 1 when on hand is critically below safety stock, else 0 |

### `service_levels.parquet` (96 rows) — customer service by region x month

| Column | Type | Description |
| --- | --- | --- |
| `region` | string | EMEA / Americas / APAC / China |
| `month` | date | first-of-month |
| `service_pct` | float | order-fill / customer service level |
| `backorder_units` | int | units on backorder in the month |

## Embedded narrative — Paints EMEA Q2 2026 service shock

The data is engineered so one signal stands out cleanly against otherwise
stable, low-noise baselines. This is the SCM-side cause that ties to the
finance **Paints-EMEA Q2 margin** story (a supply/service issue contributing to
the margin miss).

**What happened (the story the data supports):**

- The Rotterdam-origin EMEA lane **`Rotterdam-NL->EMEA-DACH`** suffers a transit
  disruption in Q2 2026. Its `lead_time_days` steps up from a steady **5 days**
  to **9 days** (ramp: Apr 7 -> May 9 -> Jun 8). The `lanes` table records the
  current elevated value (9).
- This lead-time blowout drags lane **OTIF down to ~89% in May 2026**, from a
  steady **~96% baseline**, recovering only partially to ~93% in June.
- Compounding it, two **key EMEA Decorative ("Paints") SKUs** (e.g. `DEC-1000`
  Interior Matt Emulsion) hit a **stockout at Rotterdam in May 2026**:
  `stockout_flag = 1`, `days_of_supply ~= 1` (well below safety stock). Their
  `in_full` performance collapses, which is what pulls the lane OTIF below the
  lane-wide ~91-92%.
- **EMEA `service_pct` dips to ~90.5% in May 2026** (from ~97% baseline) and
  **`backorder_units` spike to ~2,250** (vs ~90-120 baseline).

**Why it's a clean signal:** all other lanes, plants, and regions carry only
small noise and stay at ~96% OTIF / ~97% service through May 2026, so EMEA-May
is unambiguous.

**Verified figures (seed = 43):**

| Metric | Baseline (Dec 2025 - Mar 2026) | May 2026 | Jun 2026 |
| --- | --- | --- | --- |
| `Rotterdam-NL->EMEA-DACH` OTIF (volume-weighted) | 95.9% | **88.9%** | 93.0% |
| EMEA `service_pct` | ~97% | **90.6%** | 94.5% |
| EMEA `backorder_units` | ~94 | **2,258** | ~850 |
| Key EMEA SKUs stockout flags (Rotterdam, May) | 0 | **2** | 0 |
| Other-region OTIF (May 2026) | — | 95.8-96.4% | — |
